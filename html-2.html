Titulo de nuestro documento "h1"

Nota que solo se debe usar un "h1" por pagina. Puede utilizar varias etiquetas del tipo "h2" a "h6".

Los párrafos deben colocarse dentro de las etiquetas "p". Esto es necesario para que html agregue un espacio entre los párrafos. Se puede usar la etiqueta
"br" para poner un espacio adentro el párrafo.

Tipos de texto (Subtitulo del articlo "h2")

No te preocupes por el formato exacto ahora. Personalizaremos el formato mediante el uso de CSS en lecciones futuras.

La etiqueta "b" (bold en ingles) se usa para hacer letras en negrita. También puedes utilizar la etiqueta "strong" (fuerte en inglés) para poner el texto en negrita y enfatizarlo en los resultados de búsqueda web.

La etiqueta "i" se utiliza para letras en cursiva. Lo hice aquí dentro de una etiqueta "i" que está dentro de una etiqueta "b" para que se combinen.

Citas

Puede utilizar la etiqueta "blockquote" cuando desee citar un libro o articulo en su página. Utilice la etiqueta "cite" para citar qué artículo o libro está citando.
